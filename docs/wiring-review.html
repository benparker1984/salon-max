Salon Max Wiring Test Sheet

Goal: identify the correct command path and the correct feedback path for a future Raspberry Pi interface.

Working assumption

  • Yellow/Blue + Red/Green are the likely command / relay drive wires.
  • Black + White are the strongest candidate for feedback / status.
  • Do not use the 230V side for Raspberry Pi feedback.
  • Do not connect unknown 12V / 18V bed wiring directly to Pi GPIO.

Hand diagram

Measure this for every state

  • Yellow/Blue to Red/Green: voltage across pair and continuity across pair
  • Black to White: voltage across pair and continuity across pair
  • Relay coil side voltage
  • Relay contact side voltage
  • Observed behaviour: fans, lamps, start, stop, emergency stop, cooldown
